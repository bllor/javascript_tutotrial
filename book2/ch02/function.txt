3.매개변수와 인수 사용하기
function a(매개변수){
    
}
a(인수)
인수 : 함수를 호출할 때 넣는 값
매개변수 : 함수를 선언할 때 사용되는 값
인수가 매개 변수보다 많을 경우, 매개변수만큼 인수가 들어가고,
인수가 매개변수보다 적을 경우, 인수와 매칭이 되지 않은 매개변수는 undefined으로 나온다.
