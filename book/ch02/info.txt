자료형 : 데이터의 종류

1.문자열
문자들이 하나 이상 나열된 것
[1]문자열을 나타내는 방법
1)''
2)""
3)``
문자열을 ''또는 ""로 사용하고 싶으면 위의 세가지 말고 다른 것으로 감싸면 된다.
*이스케이핑
\를 붙혀서 기존의 문자와 다르게 해석되도록 하는 행위
'\'할 경우 출력되지 않고, \를 출력하고 싶으면 \\를 해야함
'\n'을 할경우 줄바꿈이 된다.

2.숫자
숫자는 따움표로 감싸지 않고 그대로 적으면 된다.
(따움표를 붙이게 되면 문자로 된다.)
parseInt('문자열')은 문자열을 숫자로 변경해주며, 여러 진법의 수를 같이 사용할 수 있다.
parseInt('111',2)=>8
Number('숫자')처럼 숫자가 따움표로 되어있는 경우에는 숫자로 반환해주지만,
만약 문자가 있을 경우 NaN(Not a Number)으로 출력이 된다.
NaN은 숫자이다.

부동소수점 문제
0.1+0.2=0.3일 것 같지만, 0.300000000004처럼 값이 이상하게 출력될 때가 있는데,
이럴 경우엔 모든 수를 정수로 변경해준 다음, 마지막에 다시 실수로 변경하는 방법이 있다.

==와 ===의 다른점
===는 값과 자료형의 비교까지 같이 한다.

논리식을 사용할 때 그리고는 && 또는 ||로 표현한다.

*
논리 연산자 사용 시 유의할 점
5&&4 ->4
''&&6 -> ''
&&연산자는 앞에 나오는 값이 참이면 뒤에 나오는 값을 결과로 보내고, 앞에 나오는 값이 거짓인 값이면 앞에 나오는 값 그대로 결과로 보냅니다.
||연산자는 앞에 나오는 값이 참인 값이면 앞에 나오는 값을 그대로 결과로 보내고, 앞에 나오는 값이 거짓인 값이면 뒤에 나오는 값을 결과로 보냅니다.

??는 널 병합 연산자로, a??b에서 a가 null이거나 undefined면 뒤에 나오는 값을 결과로 보내고, 그렇지 않을 경우 앞의 값을 보냅니다.
